# SPEC

A Python terminal-based assistant that stores and retrieves user information using a local JSON memory system.

## Features
- Command-line input
- Stores user data locally
- Retrieves saved data

## Current Functions
- new User - add a new user
- remove user - remove a user
- lights on
- lights off
